## Quick start

### One-time setup

```bash
# 1. Install Flutter (3.24+)
# https://docs.flutter.dev/get-started/install

# 2. Install Android Studio + Xcode + CocoaPods

# 3. Configure your dev machine
cd mobile
cp android/local.properties.example android/local.properties
# Edit local.properties with YOUR sdk.dir and flutter.sdk paths

# 4. Install dependencies + generate code
flutter pub get
dart run build_runner build --delete-conflicting-outputs

# 5. (Optional) Add Firebase configs:
#    Android: place google-services.json in android/app/
#    iOS:     place GoogleService-Info.plist in ios/Runner/
```

### Run

```bash
flutter run --dart-define=CDN_BASE=https://resources.positiveness.club/heal \
            --dart-define=POCKETBASE_URL=https://pocketbase.scaleupcrm.com
```

### Build release

```bash
# Android APK
flutter build apk --release \
  --dart-define=CDN_BASE=https://resources.positiveness.club/heal

# iOS (requires macOS + Xcode + signing)
flutter build ios --release \
  --dart-define=CDN_BASE=https://resources.positiveness.club/heal

# App Bundle for Play Store
flutter build appbundle --release \
  --dart-define=CDN_BASE=https://resources.positiveness.club/heal
```

## Stack

- **Flutter 3.24+** (Dart 3.5+)
- **flutter_riverpod 2** вЂ” state (matches /web pattern of single Notifier)
- **go_router 14** вЂ” deep-linkable navigation
- **drift** вЂ” local storage (settings + history), drift_flutter for native
- **pocketbase** вЂ” same backend as /web (`HEAL_*` collections)
- **firebase_core + firebase_auth + cloud_firestore** вЂ” same identity layer as /web
- **audioplayers** + **record** вЂ” same audio stack as /web
- **google_fonts** вЂ” Inter (body) + Cormorant Garamond (display)
- **flutter_animate** вЂ” micro-animations

## Content parity with /web

This app reads the **same PocketBase collections** as the web app
(`HEAL_meditations`, `HEAL_praise`, `HEAL_prayers`, `HEAL_essays`,
`HEAL_breathwork`, `HEAL_scriptures`, `HEAL_quotes`).

All media (illustrations + audio) is served from the Cloudflare-fronted
CDN at `https://resources.positiveness.club/heal/...` вЂ” no large assets
shipped in the bundle.

## CI

Android APK is built on every push to main via
`.github/workflows/mobile-build.yml`. iOS requires a self-hosted
runner with macOS + a provisioning profile (we don't have one yet).
